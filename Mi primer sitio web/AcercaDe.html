Este es un sitio tiene fines educativos, el cual ejemplifica lo planteado en la presente tesis "Manual escrito Como crear una página web". Tiene como objetivos enseñarle al lector el uso correcto del lenguaje HTML, con la finalidad de que sea capaz de crear una página web empleando el programa Dreamweaver mediante código y la interfaz gráfica.

 

Mi primer sitio web está integrado por:

  • Encabezado.

  • Menú (Home, Acerca de y Lorem Ipsum)

  • Tabla sin bordes, la cual se utilizó para el menú.

  • Hipervínculos internos y externos.

  • Imagenes.

  • Imagen de sustitución con hipervinculo externo.

  • Video.

  • Fondo.

  • Hoja de estilo en cascada.
